# Portfolio

This is the source code for a personal portfolio site built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting started

Requirements: Node.js and npm.

```sh
npm install
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build

```sh
npm run build
```
